# Simple Nodejs App
Initialize Project after create `package.json`
```
npm init --yes
```

## Install Dependencies
```
npm install express
npm install eslint
```

## Install App
```
npm i
```
Install specific package
```
npm i request@2.60.0
```


## List Packages
```
npm ls
npm ls --depth 0
```

## Check Outdated Packages
```
npm outdated
```

Update One
```
npm up eslint
```

Uninstall One
```
npm un eslint
```


Update all
```
npm up
```

## Audit Packages
Scan for vulnerabilities
```
npm audit
```

Fix Issues
```
npm audit fix
npm audit fix --force
```
