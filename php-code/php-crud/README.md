# PHP CRUD
Test Practice of basic PHP CRUD Application

## Start App
```
docker-compose up
```

Create DataBase
```
docker container exec -it mysqldb bash
mysql -u sqluser -psqluserpassword

use poc_db;

```



* Phpmyadmin: http://localhost:8080/index.php
* CRUD App: http://localhost:8000/index.php
